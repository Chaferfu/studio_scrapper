# studio_scrapper

## usage

### parameters

- week_offset int
- week_only bool
- min_duration int
- min_hour

### command line

ex: >python3 scrap.py --week_offset 1 --week_only --min_hour 19 --min_duration 3

### as application

ex: >python3 app.py

Go to: `http://127.0.0.1:5000/?week_offset=1&min_duration=3&min_hour=18&week_only=true`
